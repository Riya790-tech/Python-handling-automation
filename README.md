## 📌 Overview

This project demonstrates **file handling and automation in Python**.
It covers working with **TXT and CSV files**, along with automating common file operations.

---

## 🚀 Features

* 📄 Read & write **TXT files**
* 📊 Read & write **CSV files**
* 🔄 Rename files automatically
* 📂 Move files between folders
* ❌ Delete files safely
* ⚠️ Error handling using `try-except`

---

## 🛠️ Tech Stack

* **Python 3**
* Libraries used:

  * `os`
  * `shutil`
  * `csv`

---

## ▶️ Getting Started

### 1️⃣ Clone the Repository

### 2️⃣ Run the Program

```bash
python main.py
```

---

## 📖 Usage

### 🔹 TXT File Handling

* Write data → `write()`
* Read data → `read()`

### 🔹 CSV File Handling

* Write → `csv.writer`
* Read → `csv.reader`

### 🔹 File Operations

* Rename → `os.rename()`
* Move → `shutil.move()`
* Delete → `os.remove()`

### 🔹 Error Handling

* Uses `try-except` to handle:

  * Missing files
  * Invalid paths
  * Permission errors
